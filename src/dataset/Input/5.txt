# Protocol 5

### Knowledge

A : A, B, Kab
B : A, B, Kab
where Kab is a pre shared symmetric key
### Actions

A → B : senc((A,B,N,h(N)), Kab)
B → A : senc(M, Kab)
A → B : senc(h(M), Kab)
where N is freshly generated by A, M is freshly generated by B, h is a hash function.

### Property

Aliveness of B
lemma aliveness:
"All a b t #i.
Commit(a,b,t)@i
==> (Ex id #j. Create(b,id) @ j)"
